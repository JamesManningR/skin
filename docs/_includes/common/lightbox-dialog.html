A lightbox dialog is a child window spawned by the main web page or application. This window forms a new background layer in our design system topography.

A lightbox must remain in a hidden state for all users and devices until called upon.

Fading Lightbox

Any lightbox can be faded in and out, using the lightbox-dialog__window--fade and lightbox-dialog--mask-fade modifiers.

The default fade duration is 16ms.
